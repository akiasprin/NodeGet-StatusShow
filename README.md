# NodeGet-StatusShowR2

一个服务器状态展示页，修改自 NodeGet-StatusShow

## 开发

```bash
npm i
npm run dev
```

## 一键部署
一键部署需要主控的版本在0.2.6以上，请先到[控制面板](https://dash.nodeget.com/#/dashboard/node-manage?tab=servers)查看主控版本
